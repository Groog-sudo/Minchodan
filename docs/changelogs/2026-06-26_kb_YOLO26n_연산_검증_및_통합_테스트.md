## 작업 제목

YOLO26n / YOLO26n-seg 사전 학습 가중치 다운로드 배치 및 3단계 실 추론 연산 기반 통합 테스트 환경 구축

---

## 변경 내용

- 공식 **YOLO26n** 및 **YOLO26n-seg**의 공개 사전 학습 가중치를 다운로드하여 지정된 모델 경로(`server/models/yolo26n/`)에 안착시키는 `download_pretrained_weights.py` 유틸리티 스크립트를 신규 구축하였습니다.
- 다운로드된 실제 가중치가 팩토리 설정에 의해 `MockDetector`/`MockSegmentor` 대신 실제 YOLO 추론 엔진(`YoloDetector`, `YoloSegmentor`)으로 자동 핫스왑 및 로딩되는지 연산 무결성을 검증하는 `verify_pretrained_weights.py` 스크립트를 신규 구축하였습니다.
- 실제 YOLO26n 모델 추론, ByteTrack 추적, 이중 게이트(Reflex/Surface Gate) 분기, Redis Streams 발행 및 Redis 미연동 시의 예외 바이패스 가드레일 작동 여부를 포괄하여 3단계 전체 파이프라인의 실 연산 기능을 종합 판정하는 `integration_test_pipeline.py` 통합 테스트 스크립트를 신규 구축하였습니다.
- 보행이론 인사이트 분석 문서(`behavior_and_risk_insight.md`) 추가에 따른 프로젝트 공식 문서 인덱스(`docs/README.md`) 및 독해 순서 갱신을 동기화하였습니다.
