## 작성 양식

새 변경 사항을 기록할 때는 아래 양식을 따릅니다.

### 파일명

```
YYYY-MM-DD_[이니셜]_[작업요약].md
```

예시: `2026-06-25_dg_websocket_handshake.md`

### 파일 내용

[TEMPLATE.md](TEMPLATE.md)를 복사하여 사용합니다.
