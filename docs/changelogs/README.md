# 변경 사항 기록 (Changelogs)

> **작성일**: 2026-06-26
> **버전**: v2.0.0
> **기준**: 팀원별 단일 파일 누적 기록 방식으로 전환

---

## 개요

이 폴더는 **Minchodan** 프로젝트의 팀원별 작업 내역 및 변경 사항을 기록합니다.
기존의 날짜별로 파일을 개별 생성하던 방식에서 발생하던 파일 누적 문제를 해결하기 위해, 팀원별로 지정된 단일 파일에 날짜별 변경 사항을 누적하여 기입하는 방식으로 변경되었습니다.

---

## 작성 및 관리 규칙

- **단일 파일 관리**: 각 팀원은 자신의 이니셜이 파일명인 단일 마크다운 파일(예: `kb.md`)에 변경 사항을 누적하여 기록합니다.
- **최신 내용 상단 배치**: 새로운 변경 사항을 기록할 때는 파일의 기존 내용보다 위(상단)에 새로운 날짜의 섹션을 추가하여 최신 정보가 가장 먼저 보이도록 합니다.
- **템플릿 준수**: 변경 사항을 작성할 때는 [TEMPLATE.md](TEMPLATE.md)의 날짜별 세션 형식을 복사하여 사용합니다.

## 작성 양식

자세한 작성 형식은 [TEMPLATE.md](TEMPLATE.md)를 참조하십시오.
새로운 변경 사항 기록 시, 해당 팀원의 파일 상단에 템플릿의 날짜 섹션을 복사하여 붙여넣고 내용을 작성하시면 됩니다.
