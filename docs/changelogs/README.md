## 기록 방식

### 자동 기록 (스크립트 사용)

아래 명령을 프로젝트 루트(`./Minchodan`)에서 실행하면 본인 파일에 작업 로그가 자동으로 append됩니다.

| OS | 명령 |
| --- | --- |
| **Windows (PowerShell 또는 cmd)** | `scripts\postwork.bat` |
| **macOS / Linux (bash 또는 zsh)** | `bash scripts/postwork.sh` |

### 수동 기록

본인 파일 하단에 아래 양식의 엔트리를 추가합니다.

```markdown
---

### YYYY-MM-DD | [단계]단계 | [작업 요약]

- **커밋**: `[커밋 접두어]: [커밋 설명]`
- **변경 내용**:
  - [변경 내용을 한 줄씩 작성합니다.]
- **관련 파일**: `파일1`, `파일2`, ...
- **검증 결과**: [테스트 통과 여부, KPI 달성 여부]
```

상세 양식은 [TEMPLATE.md](TEMPLATE.md)를 참조합니다.
